* Upgrade on macOS

** [[https://mac.install.guide/homebrew/4][Update Homebrew]]

#+BEGIN_SRC shell
brew update
brew doctor
# to upgrade packages
brew upgrade
#+END_SRC

** [[https://docs.npmjs.com/try-the-latest-stable-version-of-node][Install the latest node]]

#+BEGIN_SRC shell
brew upgrade nvm
nvm install node
#+END_SRC
* Cleanup (versions)
Versions are stored in ~/.nvm/versions; there doesn't appear to be a command to prune old versions.

#+BEGIN_SRC
  rm -rf ~/.nvm/versions
  nvm install node
#+END_SRC

* nvm cheat sheet
- nvm ls :: list install versions
- nvm install node :: install lastest stable version of node

** Troubleshooting
