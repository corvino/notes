** To find the identifier

** To format in FAT32
sudo diskutil list
sudo diskutil eraseDisk FAT32 [NAME] MBRFormat /dev/disk#
    e.g., 
** To unmount, if already FAT32 but mounted
diskutil unmountDisk /dev/disk<#>
** To flash an image
cat image.img | sudo dd of=< drive address > bs=32m'
sudo dd bs=1m if=<file.img> of=/dev/rdisk<#> conv=sync
or maybe
sudo dd if=ubuntu-19.10-live-server-arm64.iso of=/dev/rdisk4

curl -O https://raw.githubusercontent.com/hypriot/flash/2.3.0/flash
