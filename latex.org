** Non-free fonts
See [[https://tug.org/fonts/getnonfreefonts/][getnonfreefonts]]

#+BEGIN_SRC bash
curl --remote-name https://www.tug.org/fonts/getnonfreefonts/install-getnonfreefonts
texlua install-getnonfreefonts

getnonfreefonts --sys garamondx
# Migh need dependencies; see "Check on missing package"
tlmgr install fontaxes
tlmgr install ly1
#+END_SRC
** Set Helvetica as default font
#+BEGIN_SRC latex
\usepackage[scaled]{helvet}
\renewcommand\familydefault{\sfdefault}
\usepackage[T1]{fontenc}
#+END_SRC
** Set garamondx as the default font
#+BEGIN_SRC latex
\usepackage{garamondx}
#+END_SRC
