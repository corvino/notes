* Commands and shorcuts
|---------+-----------------------------------------------|
| C-zr    | Run SML                                       |
| C-cC-l  | loads file into repl, defaults to buffer file |
| C-cC-r  | loads region into repl                        |
| C-cC-b  | loads buffer into repl                        |
| C-c-C-s | switch to repl                                |
|---------+-----------------------------------------------|
* To run a program without the repl, send the file to snadard input
sml < prog.sml
